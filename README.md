# The Adventure of Mammy

GAMEPLAY PROTOTYPE

The Adventure of Mammy is a solo-developed 2D action prototype created to explore combat, movement, and character controller systems in Unity. The project serves as a technical sandbox for testing gameplay mechanics commonly found in action-platformers, Metroidvanias, and Soulslike-inspired games.

## Prototype Features

### Implemented
- Player movement and traversal
- Directional melee attacks
- Health and stamina systems
- Healing mechanics
- Character animation and state management

### Experimental Systems
- Combat tuning and encounter design
- Enemy behavior prototyping
- Parry and defensive mechanics
- Environmental interaction concepts

---

## Purpose

This prototype was created as a technical and design exploration project focused on gameplay feel, combat responsiveness, and system architecture. It is used to experiment with mechanics, iterate on player feedback, and test new gameplay ideas before committing them to larger projects.
